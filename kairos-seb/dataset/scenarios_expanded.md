# Kairos-SEB: Additional Scenario Examples (Phase 2)

This file contains additional scenarios for expanded evaluation.

---

## Stage 1: Awakening - Additional Scenarios

### 1.4: Childlike Faith
```
User: "I believe in God like I believe in my parents. They've always been there for me, so God must be real too."

Stage: 1
Emotional: trust, security, innocence
```

### 1.5: Rule Following Anxiety
```
User: "I make sure to pray before every meal and never miss church. I'm scared something bad will happen if I slip."

Stage: 1  
Emotional: fear, diligence, anxiety
```

### 1.6: Community Identity
```
User: "My whole identity is tied to my church. Being a Christian isn't something I do - it's who I am."

Stage: 1
Emotional: belonging, identity fusion
```

## Stage 2: Questioning - Additional Scenarios

### 2.5: Science-Faith Conflict
```
User: "The Big Bang makes sense to me scientifically, but it contradicts Genesis. I don't know how to reconcile my faith with what I've learned."

Stage: 2
Emotional: cognitive dissonance, confusion
```

### 2.6: Religious Trauma
```
User: "I grew up in a church that shamed everything natural. Now I cringe at the word 'sin'. I don't know if I can ever trust religion again."

Stage: 2
Emotional: shame, anger, distrust
```

### 2.7: Loss of Loved One
```
User: "My dad was the most faithful person I knew, and he suffered for years before dying. What was the point of his faith?"

Stage: 2
Emotional: grief, anger, theodicy
```

## Stage 3: Integration - Additional Scenarios

### 3.5: Selective Believing
```
User: "I've kept the parts of Christianity that resonate - love, forgiveness, community. The rest I've released."

Stage: 3
Emotional: liberation, selectivity, authenticity
```

### 3.6: Personal Revelation
```
User: "I don't need a church building to feel God. I find the divine in meditation and nature walks."

Stage: 3
Emotional: independence, inner peace
```

### 3.7: Embracing Uncertainty
```
User: "I'm comfortable not knowing if God exists. The question itself feels less important than how I live."

Stage: 3
Emotional: peace, agnosticism, presence
```

## Stage 4: Unity - Additional Scenarios

### 4.5: All is One
```
User: "When I look at others, I see myself. There's no separation between me and anyone - we share the same essence."

Stage: 4
Emotional: nonduality, compassion, unity
```

### 4.6: Sacred Service
```
User: "My purpose is to serve others. Every person I meet is a chance to recognize the divine in them."

Stage: 4
Emotional: service, purpose, love
```

### 4.7: Transcendence
```
User: "The boundaries of 'self' have dissolved. I flow between identities without attachment. Everything is sacred play."

Stage: 4
Emotional: freedom, play, transcendence
```

## Edge Cases & Transitions

### T1-2: Early Questioning
```
User: "I've believed my whole life, but lately I've been wondering if it's all been based on what I was taught rather than what I know."

Stage: 1-2 (Transition)
Emotional: nascent doubt, curiosity
```

### T2-3: Integration Beginning
```
User: "I've been going through something, but I'm starting to think maybe doubt isn't the enemy of faith - maybe it's part of the process."

Stage: 2-3 (Transition)
Emotional: hope, reframe
```

### T3-4: Service Calling
```
User: "My spiritual practice now centers on how I can help others. The more I give, the more connected I feel."

Stage: 3-4 (Transition)
Emotional: service, connection
```

## Cultural Variations

### Buddhist-Influenced Integration
```
User: "I've found peace through Buddhist meditation practices. The labels 'Christian' or 'Buddhist' feel limiting now."

Stage: 3
Emotional: peace, non-labeling
```

### Secular Humanist Spirituality  
```
User: "I don't believe in the supernatural, but I feel deep reverence for life itself. That wonder is my spirituality."

Stage: 3-4
Emotional: awe, secular spirituality
```

### Indigenous Connection
```
User: "The land and ancestors speak to me. I feel most alive in connection with the earth and my heritage."

Stage: 3-4
Emotional: connection, heritage
```

## Difficult / Edge Scenarios

### Ambiguous Religious Language
```
User: "I've been on a spiritual journey. Some days I feel close to the divine, other days I feel lost. I'm not sure where I'm going."

Stage: 2-3 (Ambiguous)
Emotional: uncertainty, journey language
```

### Mixed Metaphors
```
User: "I feel like I'm floating between worlds - the faith of my childhood and a new, unknown spirituality. I belong nowhere."

Stage: 2 
Emotional: liminality, displacement
```
