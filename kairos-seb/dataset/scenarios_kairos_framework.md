# Kairos-SEB: Framework-Specific Scenarios

This document contains scenarios aligned with specific Kairos framework subgroups and concepts for deeper evaluation.

---

## Category 1: Enneagram Types (9 Types)

Based on the RHETI Enneagram with first-person statements

### Type 1 - The Reformer (Perfectionist)
```
User: "I need things to be right - not just okay. When something's off, I can't rest until it's corrected. My inner critic is relentless."

Stage Context: Awakening (rule-following) → Questioning (when standards fail)
Emotions: frustration, guilt, self-judgment
Framework: Enneagram Type 1
```

### Type 2 - The Helper (People-Pleaser)
```
User: "I naturally put others first, sometimes to my own detriment. I secretly fear that if I don't help, I'm not worthy of love."

Stage Context: Awakening (dependence) → Questioning (when giving becomes draining)
Emotions: resentment, martyrdom, hidden pride
Framework: Enneagram Type 2
```

### Type 3 - The Achiever (Performer's Anxiety)
```
User: "My worth feels tied to achievement. When I succeed, I feel valuable. When I fail, I feel like nothing. I've achieved so much but wonder if anyone's ever seen the real me."

Stage Context: Awakening (performance-based identity) → Questioning (identity crisis)
Emotions: shame, exhaustion, fraudulence
Framework: Enneagram Type 3
```

### Type 4 - The Individualist (Creative Sensitivity)
```
User: "I've always felt different - like I'm missing something others have. I create an idealized image because the real me feels flawed."

Stage Context: Integration (embracing uniqueness) or Questioning (never feeling enough)
Emotions: envy, melancholy, shame
Framework: Enneagram Type 4
```

### Type 5 - The Investigator (Detached Observer)
```
User: "I withdraw to understand. Too much emotion feels overwhelming. I'm fine being alone - it gives me space to think deeply."

Stage Context: Awakening (isolation) or Integration (healthy boundaries)
Emotions: withdrawal, confusion, avarice
Framework: Enneagram Type 5
```

### Type 6 - The Loyalist (Anxiety-Driven)
```
User: "I scan for threats and question authority. I want to trust but keep looking for reasons not to. Security feels hard-won."

Stage Context: Questioning (skepticism) → Integration (learned trust)
Emotions: anxiety, suspicion, fear
Framework: Enneagram Type 6
```

### Type 7 - The Enthusiast (Fear of Missing Out)
```
User: "I keep moving to avoid feeling pain. I'm always planning the next adventure because the present feels incomplete somehow."

Stage Context: Awakening (avoidance) → Integration (facing the void)
Emotions: anxiety, denial, planning
Framework: Enneagram Type 7
```

### Type 8 - The Challenger (Control Issues)
```
User: "I'm strong and protective but can become controlling. I learned to never show vulnerability - it felt unsafe. Behind the strength is tender vulnerability I'm afraid to reveal."

Stage Context: Awakening (control) → Integration (vulnerability)
Emotions: anger, fear, protectiveness
Framework: Enneagram Type 8
```

### Type 9 - The Peacemaker (Narcotization)
```
User: "I tune out conflict to stay peaceful. Sometimes I realize I've been agreeing when I actually disagree. My own needs get lost in keeping the peace."

Stage Context: Awakening (passivity) → Integration (asserting self)
Emotions: frustration, self-neglect, merge
Framework: Enneagram Type 9
```

## Category 2: IFS (Internal Family Systems)

Based on parts work and Self leadership

### Meeting a Protector Part
```
User: "A part of me aggressively fixes problems before they happen. I thought it was just being proactive, but it's actually terrified of failure."

Stage Context: Questioning (observing parts)
Emotions: anxiety, protective, fear
Framework: IFS - Protector Part
```

### Accessing Self
```
User: "When I'm with my angry part and can just notice it without being it - I feel calm, curious, compassionate. That's not the anger talking."

Stage Context: Integration
Emotions: Self-energy (8 Cs)
Framework: IFS - Self leadership
```

### Young Part Integration
```
User: "There's a young part of me that still feels unsafe when people raise voices. I want to tell it: you're safe now. I can protect you."

Stage Context: Integration
Emotions: care, protection, reparenting
Framework: IFS - Inner Child
```

### Polarized Parts
```
User: "One part wants to stay safe and small. Another part wants to take risks and be seen. They're always fighting. When I ask each what it fears, the fear is the same: rejection."

Stage Context: Questioning → Integration
Emotions: conflict, fear
Framework: IFS - Polarized Parts
```

## Category 3: Somatics & Body-Based

### Somatic Awareness
```
User: "I notice my breath catches when I talk about my father. My chest gets tight. My body knows things before my mind catches up."

Stage Context: Integration
Emotions: somatic awareness
Framework: Somatics - Body awareness
```

### Chakra/Energy Release
```
User: "During meditation, heat rose through my spine and I started crying. It was like something was finally releasing after being stuck for years."

Stage Context: Integration→Unity
Emotions: release, catharsis
Framework: Somatics - Energy clearing
```

### Nervous System Regulation
```
User: "I used to live in constant fight-or-flight. Now I notice the early signs - shallow breathing, tension in my shoulders. I can regulate before I react."

Stage Context: Integration
Emotions: regulation, self-awareness
Framework: Somatics - Nervous system
```

## Category 4: Buddhist Psychology

### Vipassana - Observing Sankaras
```
User: "I notice my anger arises automatically - like a preset response. When I observe it without reacting, it passes through like a wave."

Stage Context: Integration
Emotions: observation, equanimity
Framework: Buddhist - Vipassana
```

### Liberation from Automatic Reactions
```
User: "The pattern used to run me completely. Now there's space between trigger and response. It's like waking up from a dream - I have choices I didn't know I had."

Stage Context: Integration → Unity
Emotions: freedom, choice
Framework: Buddhist - liberation
```

### Non-Duality
```
User: "I used to think there was a 'me' and an 'other.' Now that distinction feels artificial. There's just experience flowing."

Stage Context: Unity
Emotions: nonduality, release
Framework: Buddhist - non-self (anatta)
```

## Category 5: Conscious Leadership

### Stage 1 - Command & Control
```
User: "I led through authority - my way or the highway. Results came but people burned out. I couldn't understand why they couldn't keep up."

Stage Context: Awakening
Emotions: frustration, control, results-focused
Framework: Conscious Leadership - Stage 1
```

### Stage 2 - Expertise & Merit
```
User: "I thought if I was the smartest person in the room, people would follow. But expertise doesn't build loyalty - connection does."

Stage Context: Questioning
Emotions: realization, humility
Framework: Conscious Leadership - Stage 2
```

### Stage 3 - Coaching & Development
```
User: "My role shifted to developing others. The more I empower them to lead, the less I need to. It's been a humbling journey."

Stage Context: Integration
Emotions: service, empowerment
Framework: Conscious Leadership - Stage 3
```

### Stage 4 - Guiding Vision
```
User: "I don't need to be the smartest or loudest anymore. I hold the vision and help others find their voice. The impact is so much bigger."

Stage Context: Unity
Emotions: vision, service
Framework: Conscious Leadership - Stage 4
```

## Category 6: Human Design Types

### Generator (Energy Type)
```
User: "I'm built to do and create. When I'm engaged in meaningful work, I feel alive. When I'm stuck in meetings, I'm drained. I need to follow my gut."

Stage Context: Integration
Emotions: vitality, frustration when blocked
Framework: Human Design - Generator
```

### Manifesting Generator (Quick Start)
```
User: "I move fast and get frustrated by process. I'm great at starting but sometimes burn out because I don't listen to my body before diving in."

Stage Context: Questioning → Integration
Emotions: energy, impatience
Framework: Human Design - Manifesting Generator
```

### Projector (Strategic Guide)
```
User: "I'm designed to guide, not generate. Before I understood this, I burned out trying to do what generators do. Now I pace myself and wait for invitations."

Stage Context: Integration
Emotions: recognition-seeking, wisdom
Framework: Human Design - Projector
```

### Reflector (Sampling)
```
User: "I take time to absorb before decisions. My clarity comes through waiting and watching. It's different from others but it's my design."

Stage Context: Integration
Emotions: patience, clarity through timing
Framework: Human Design - Reflector
```
